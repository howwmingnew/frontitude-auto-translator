   - **Empty state**: If `!cached || !cached.matches || cached.matches.length === 0`, render `<div class="context-panel-empty">` with `App.t('contextPanelEmpty')` and return early (still include translation fields below)
   - **Code snippet section**: Render first match only: `<div class="context-panel-snippet">`, file path as `<div class="context-panel-file">&#128196; {escaped file}:{line}</div>`, code as `<pre class="context-panel-code">{escaped snippet}</pre>`. If `cached.matches.length > 1`, add `<button class="context-panel-more" data-key="{key}">` with `App.t('contextPanelMore', cached.matches.length - 1)` text
   - **AI description section**: `<div class="context-panel-description">`. If `cached.description` exists, show `<span class="context-panel-desc-icon">&#128161;</span> {escaped description}`. Otherwise show `<span class="context-panel-no-desc">{App.t('contextPanelNoDesc')}</span>`
   - **Translation fields**: Get `App.getState().jsonData`, find target languages (`Object.keys(jsonData).filter(k => k !== 'en').sort()`). For each lang, render `<div class="context-panel-lang-row">` with `<label>{LANG.toUpperCase()}</label>`, `<input type="text" class="context-panel-input" data-lang="{lang}" data-key="{key}" value="{escaped value}">`, `<button class="context-panel-save" data-lang="{lang}" data-key="{key}">{App.t('cellEditSave')}</button>`
   - Use `App.escapeHtml()` for ALL dynamic content. For input values, also replace `"` with `&quot;`

2. **`expandContextPanel(keyRow, key)`** function:
   - Call `collapseCurrentPanel()` first to close any open panel
   - Get column count from `keyRow.children.length`
   - Create `tr` element with class `context-panel-row`
   - Create `td` element with `colSpan = colCount` and class `context-panel-cell`
   - Get cached data: `var cached = App.getContextCache().get(key)`
   - Set `td.innerHTML = buildPanelHTML(key, cached)`
   - Append td to tr, insert tr after keyRow: `keyRow.parentNode.insertBefore(tr, keyRow.nextSibling)`
   - `App.setState({ expandedPanelKey: key })`
   - Add `expanded` class to keyRow for visual feedback: `keyRow.classList.add('context-panel-expanded')`

3. **`collapseCurrentPanel()`** function:
   - Find existing panel: `var existing = App.dom.editorTable.querySelector('.context-panel-row')`
   - If exists, remove the `context-panel-expanded` class from the previous sibling row: `existing.previousElementSibling && existing.previousElementSibling.classList.remove('context-panel-expanded')`
   - Remove the panel row: `existing.parentNode.removeChild(existing)`
   - `App.setState({ expandedPanelKey: null })`

4. **`toggleContextPanel(keyRow, key)`** function:
   - `var s = App.getState()`
   - If `s.expandedPanelKey === key`, call `collapseCurrentPanel()` (toggle off)
   - Else call `expandContextPanel(keyRow, key)`

5. **Panel event delegation** — Add a click handler on `App.dom.editorTable` for panel interactions (or add inside `expandContextPanel`). Use event delegation:
   - If click target matches `.context-panel-save` button: read `data-lang` and `data-key` attributes, find the sibling input with matching `data-lang` and `data-key`, get its `.value`.    - If click target matches `.context-panel-more` button: find the `.context-panel-snippet` parent, get cached data via `App.getContextCache().get(key)`, render ALL matches (not just first), replace the snippet section content, hide the "more" button.
